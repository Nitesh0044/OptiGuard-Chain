## Overview
OptiGuard-Chain is a blockchain-based security project written in C++. It implements a decentralized ledger to track and audit logs with built-in threat detection scoring.

## Features
- **SHA-256 Hashing:** Unique digital fingerprints for data integrity.
- **Pointer-Based Linking:** Blocks are linked using C++ pointers for efficient memory access.
- **CMake Build System:** Cross-platform compilation support.
- **Security Audit Logs:** Tracks sender IPs and security threat scores.

## How to Build
1. Create a build directory: `mkdir build && cd build`
2. Configure CMake: `cmake ..`
3. Compile: `mingw32-make`
4. Run: `./OptiGuard_Node.exe`
